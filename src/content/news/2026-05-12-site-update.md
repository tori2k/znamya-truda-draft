За последние дни сайт fczt1909.ru получил несколько технических улучшений.

**Исправлено:**

- Кнопки на мобильных устройствах больше не обрезаются — проблема была в настройках отображения секций
- Карта стадиона на странице клуба теперь загружается корректно
- Кнопки по всему сайту стали округлёнными

**Добавлено:**

- Отчёт о матче с Сатурном-2 (0:2, 3 тур) — доступен на главной странице в блоке последнего матча
- Обновлены данные последней игры

**Яндекс.Метрика:**

Счётчик уже встроен в сайт. Чтобы он заработал, нужно одно действие: в репозитории на GitHub зайти в **Settings → Secrets and variables → Actions** и добавить секрет с именем `PUBLIC_YM_COUNTER_ID` и значением `109108973` (номер из кода счётчика). После следующего деплоя статистика начнёт собираться. Больше ничего вставлять не нужно.
